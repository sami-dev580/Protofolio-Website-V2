# Sami.Dev | Portfolio Website 🚀

Welcome to my old personal portfolio! I'm a 16-year-old developer from Tunisia 🇹🇳. 
I'm making this project open-source for anyone who wants to learn or use it as a template.

## ✨ Features
- **Fully Responsive Design:** Mobile-friendly and works on all screens.
- **Dark Mode Aesthetic:** Modern deep blue and cyan theme.
- **Interactive Navigation:** Smooth scrolling and a functional mobile burger menu.
- **Sections:** Home, About, Skills, Works, Services, and Contact.

## 🛠️ Built With
- **HTML5** & **CSS3** (Custom properties & Flexbox/Grid).
- **JavaScript** (Vanilla JS for the mobile menu logic).
- **FontAwesome** for the sleek icons.
